Empirical Software Engineering

A Empirical Software Engineering tem foco em pesquisa aplicada para engenharia de software com fortes componentes empíricos, e publicações de resultados relevantes para pesquisadores e praticantes. Os estudos geralmente envolvem análise de data e experiência que podem ser usadas para carcterizar, evaluar revelar relações entre práticas de desenvolvimento de software e tecnologias.
